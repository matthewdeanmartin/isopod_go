# isopod_go
Trivial text adventure game to make a basic go build script

## Features
Does basic compile, format, lint, docs

## Build
```bash
make all
```

## Docs
```bash
make docs
```
View at [http://localhost:6060/pkg/isopod_game/game/](http://localhost:6060/pkg/isopod_game/game/)
